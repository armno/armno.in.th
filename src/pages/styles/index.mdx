## Code

Inline code can be written like `console.log('Hello, World!')`.

Code blocks can be written with triple backticks:

```typescript
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}
```

Kbd tag can be used to display keyboard keys: <kbd>Cmd</kbd> + <kbd>k</kbd> or <kbd>Ctrl</kbd> + <kbd>k</kbd>
